# alipay_ng2

用angular2框架，模仿移动端支付宝页面


项目结构:

        └─src
            ├─app
            │  ├─allApp   (所有应用)
            │  ├─friends  (朋友)
            │  ├─home     (首页)
            │  ├─mine     (我的)
            │  ├─shared   (共享【底部footer】)
            │  └─shop     (口碑店铺)
            └─images      (图片资源)



- 确保已经装了 [node.js](https://nodejs.org/) installed（>= 6.0）

- 安装依赖: `npm install`

- 开启项目： `npm start`

- 浏览器打开： `http://localhost:3100` ( port 修改可在package.json  "scripts">"server" 修改 --port ,域名访问 --host 192.168.1.1 )


该项目所有icon图标 来自**http://www.iconfont.cn/**
